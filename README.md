# Companion

Приложение-помощник с рутиной. Android, мобильное, вертикальная ориентация.

Движок: **Unity 6000.4.9f1**, 2D URP. Целевая платформа — Android (портрет).
Данные пользователя (таймеры, таблица лекарств) сохраняются локально.

## О проекте

Companion — приложение-помощник для занятого человека.

- Создание нескольких параллельных таймеров одной кнопкой.
- Заведение карточки приёма лекарств, рассчитанной на утро, день и вечер.
- Отметка о приёме лекарства одной кнопкой.
- Скрытое накопление данных о принятых лекарствах — задел под будущий апдейт с просмотром истории.
- Удаление записи свайпом влево.
- Активный список лекарств сохраняется каждый день; в полночь отметки о приёме сбрасываются,
  чтобы список можно было использовать снова.

Приложение полностью написано Claude с использованием самостоятельно разработанных инструментов
для Unity. Я выступал в роли заказчика и тестировщика.

Было две цели:

1. Получить приложение для автоматизации рутины.
2. Проверить ранее созданный инструментарий для Claude на способность быстро собрать целое
   приложение на Unity.

Неожиданной дополнительной ценностью проекта стало создание самописного нативного плагина для
работы с уведомлениями на Android-устройствах.

## Концепция

При запуске — экран на чёрном фоне с кнопками-разделами. На старте два раздела:
**Таймер** и **Лекарства**.

## Раздел «Таймер»

- Кнопка **«Добавить»**, ниже — список ранее созданных таймеров в виде кнопок.
- Нажатие на кнопку таймера запускает его.
- По завершении таймера открывается попап и звучит звуковое сообщение.
- При создании таймера указываются: длительность в минутах и название.

## Раздел «Лекарства»

Окно с таблицей приёма препаратов, которую пользователь заполняет сам.

### Строка таблицы

Каждая строка состоит из двух частей:

1. **Шапка строки** (две колонки): `Время приёма` / `Препарат`.
2. **Подпись под строкой** (свободный текст): `Дозировка` / `Как принимать` / `Важно`.

### Управление таблицей

- Кнопка **«Добавить»** открывает попап добавления строки: время, препарат, дозировка,
  доп. информация.
- **Свайп строки влево** заменяет кнопку «Добавить» на **«Удалить»** — позволяет удалить строку.
- **Нажатие на строку** перечёркивает её как выполненную (повторное нажатие снимает отметку).
  Скролл списка при этом остаётся доступным (нажатие ≠ прокрутка).

### Пример наполнения

| Время приёма | Препарат | Дозировка | Как принимать | Важно |
|---|---|---|---|---|
| За 30 мин до завтрака | Рабепразол | 20 мг (1 капс) | Запить водой | Вскрывать оболочку нельзя |
| За 30 мин до завтрака | Висмута трикалия дицитрат | 120 мг (2 таблетки) | Разжевать или рассосать (или проглотить) | Натощак |
| После завтрака | Кларитромицин | 500 мг (1 таб) | После еды | Может быть горький вкус |
| После завтрака | Амоксициллин | 1000 мг (1 таб) | После еды | Может быть диарея |
| ЧЕРЕЗ 1 час | Энтерол (профилактика диареи) | 1 капс | С едой | Принимать весь 10-14 дневный курс |
| За 30 мин до ужина | Висмута трикалия дицитрат | 120 мг (2 таблетки) | Разжевать или рассосать | Натощак |
| После ужина | Кларитромицин | 500 мг (1 таб) | После еды | |
| После ужина | Амоксициллин | 1000 мг (1 таб) | После еды | |
| ЧЕРЕЗ 1 час | Энтерол (вечерняя доза) | 1 капс | С едой | |
